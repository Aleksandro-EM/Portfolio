CarGo - Car Rental Platform

A production-ready Spring Boot web application for online car reservations featuring secure payment processing via Stripe API and cloud-based file storage with AWS S3. The platform includes comprehensive role-based access control, image upload functionality, and a complete vehicle management system.

Key Features:
  • Role-based access control (Admin, User)
  • Secure payment integration with Stripe
  • AWS S3 for scalable image storage
  • Full CRUD operations for vehicle management
  • Deployed on Heroku with MySQL database
